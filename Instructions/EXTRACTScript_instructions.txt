Instructions on how to use EXTRACTScript.py python script.
Additionally, these instructions contain the information about EPD test data provided.


-  Selects and extracts data from Excel file containing multiple 96-well plate analyses on the same sheet. 

-  Divides each 96-well plate data to Excel files of their own, and zipps the files into a zip container stored at result_files directory. 

-  Separation of data is required for the EPDScript_96.py and DORAScript_96.py to function. In these scripts, the data file should only contain one well plate of data.



Prerequirements

-  Data file in data_files directory.


Script run

1.  Open Windows PowerShell by going into Source directory, typing "powershell" to address bar and pressing Enter.

2.  Run script by typing "python EXTRACTScript.py" on Windows PowerShell.

3.  After script run, the extracted plates are stored in results_files directory in their own Excel files. Continue data analysis from those files according to the EPDScript_instructions.txt file.



Instructions on how provided test data was run with EXTRACTScript.

Example 1: Test_multiple_EPD_96wp.xlsx

The raw data sheet contains two 96-wp of data on top of each other. To separate the two 96-wps to different Excel files, run EXCTRACTScript.py.
