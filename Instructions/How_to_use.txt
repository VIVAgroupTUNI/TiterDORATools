Follow these steps to use the script:


1.  Place the required configuration files of the scripts inside the "config_files"
    directory that comes with TiterDORATools-main.zip file if they are not there.

2.  Place all master files that are to be used with the scripts inside the "master_files"
    directory that comes with TiterDORATools-main.zip file if the master files are not there.

3.  Place all data files that are to be used with the scripts inside the "data_files"
    directory that comes with TiterDORATools-main.zip file.

4.  Run the desired script in Windows PowerShell using python to start them
    and follow along the script prompts. Access Windows PowerShell by first going into Source directory, typing powershell into address bar and pressing Enter.

5.  More detailed and further info about the script can be accessed by running the script with
    any of the following parameters ["i", "info", "h", "help"], e.g. "python EPDScript_96.py info"
    on Windows PowerShell.

6.  Start script run by typing e.g. "python EPDScript_96.py" to Windows PowerShell and press Enter.
