Instructions on how to use STATScript.py python script.
Additionally, these instructions contain the information about DORA test data provided.


-  Runs non-parametric statistical tests for selected groups either Kruskal-Wallis test with Dunn's test for pairwise comparisons or Mann-Whitney U test. 

-  P-values are corrected with Bonferroni correction. 

-  Creates new sheet into the data file to store results. 

-  Shows results in table format showing p-values and statistical significance indicated by symbols that correspond to "not significant", "0.05", "0.01", "0.001" and "0.0001" level of significance.



Prerequirements

-  Data file in data_files directory. 

-  Data file contains a sheet generated by 4PLScript.py or excel sheet formated the same way. Sheet contains the IC50/CC50/EC50 results.



Script run

1.  Open Windows PowerShell by going into Source directory, typing "powershell" to address bar and pressing Enter.

2.  Run script by typing "python STATScript.py" on Windows PowerShell. To access more instructions on how the script functions, run the script with any of the following parameters ["i", "info", "h", "help"] e.g., "python STATScript.py help" on Windows PowerShell.

3.  After script run, the results are shown in the provided raw data file.
